# Tire

## Запуск на Ubuntu
1. Установите Node.js 18+ и npm.
2. В корне проекта установите зависимости:
   `npm install`
3. Запустите сервер:
   `npm start`
4. Откройте в браузере:
   `http://localhost:3000/index.html` — форма клиента  
   `http://localhost:3000/master.html` — список заказов мастера

База данных SQLite хранится в папке `data/` (создаётся автоматически).
